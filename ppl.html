🏴‍☠️ About the League

Pirate Premier League (PPL) is a DLS (Dream League Soccer) mobile football community hosted on Discord. Tournaments are conducted seasonally — player stats are tracked and updated manually after each match. Join the server to register for the next season!

📋 How Tournaments Work

• Seasons played in DLS mobile game
• Players registered by league admin
• Stats (goals, assists, conceded) tracked manually
• Trophies awarded at end of each season
• All match data updated on this site
